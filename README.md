# Brainfu4J

Pronounced brainfudge (the 4 is silent)


## Literals

| Literal   | Explaination      | Brainfuck Character |
| --------- | ----------------- | ------------------- |
| public    | Start loop        | [                   |
| private   | End loop          | ]                   |
| class     | Pointer back      | <                   |
| interface | Pointer forward   | >                   |
| static    | Increment address | +                   |
| final     | Decrement address | -                   |
| println   | Output            | .                   |
| scanner   | Input             | ,                   |
